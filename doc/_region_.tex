Modulus:

Greatest common divisor:
gcd (a,0) = a
gcd (a,b) = gcd (b, a mod b)
Least common multiple:
lcm (a,b) = |a*b| / gcd (a,b)
Conditional Probability:
P(a|b) = P(a AND b) / P(b)
Bayes Theorem:
P(a|b) = P(b|a)P(a) / P(b)
